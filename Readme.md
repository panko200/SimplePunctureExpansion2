簡易パンク・膨張2
製作者：Panko200
配布場所：なし

## 概要

AfterEffectやIllustratorのパンク・膨張を再現しようと奮闘したものです。

## 使用方法

映像エフェクト「簡易パンク・膨張2」が追加されます。  
強度をいじることによってパンク膨張がかかります。
解析では、入力された画像から頂点を無理やり探し始めます。ほんで頂点認識されたところと辺を変形させることでパンク・膨張を再現しています。

## アンインストール方法

1. YMM4を起動して`ヘルプ(H)`>`その他`>`プラグインフォルダを開く`をクリックする。
2. YMM4を終了する。
3. `SimplePunctureExpansion2Effect`という名前のフォルダを削除する。

## 注意点
